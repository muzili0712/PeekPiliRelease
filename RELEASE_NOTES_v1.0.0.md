# Release v1.0.0

## 🎉 新版本发布

## 📝 更新内容

1. 支持Android、Windows和ios三个平台
2. json配置接口同ok影视，支持调用接口内解析
3. 具有实验室功能为网页嗅探器，输入网址自动嗅探并推送壳子播放
4. 全平台支持T4类型源
5. Android和Windows支持py源且支持本地代理(http://127.0.0.1:9979/proxy?do=py)
6. 全平台支持猫源open格式
7. 全平台支持猫源nodejs格式，Win端特别对牛二猫源接口的弹幕搜索界面适配
8. 壳自带弹幕搜索(自写算法包括：哎呦疼忙碧美韩港)
9. 弹幕支持渐变色和动态特效
