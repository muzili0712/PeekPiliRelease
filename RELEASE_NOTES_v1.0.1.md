# Release v1.0.1

## 🎉 新版本发布

## 📝 更新内容

### ✨ 新增功能

1. **不夜猫源适配支持**
   - 新增Android和iOS平台的不夜猫源适配
   - ⚠️ iOS用户注意：在两个nodejs接口之间直接切换可能导致应用闪退，建议切换步骤：先切换到非node接口 → 清理应用后台进程 → 重新打开应用 → 再进行接口切换

2. **站源搜索屏蔽功能**
   - 新增站源搜索屏蔽开关，可在站源选择界面控制各站源的搜索状态
   - 每个站源后方提供放大镜图标，点击即可切换该站源的搜索屏蔽状态
   - 默认所有站源搜索功能均为开启状态

3. **历史记录管理优化**
   - 新增历史记录单个删除功能，方便用户精确管理观看历史

4. **播放器功能增强**
   - 新增播放器应用内全屏功能（鼠标右键点击全屏按钮即可切换）
   - MPV播放器新增音轨和视轨切换功能

### 🐛 问题修复

5. **MPV播放器修复**
   - 修复Windows端MPV播放器无画面显示的问题

6. **历史记录修复**
   - 修复Windows端历史记录无法正常访问的问题

7. **弹幕修复**
   - 修复"碧"弹幕相关问题

### 🔧 兼容性改进

8. **Python源兼容性**
   - 兼容使用 `from com.whl.quickjs.wrapper import QuickJSContext` 的Python源（如3qys.py）
   - 兼容使用 `getCache`/`setCache` 方法的Python源（如LiteApple.py）

### ⚠️ 已知问题

9. **Windows 10后台进程**
   - Windows 10系统退出应用后，可能仍有后台进程残留（Windows 11无此问题）
   - 目前暂无法完全解决，Windows 10用户如遇此情况，请手动结束相关进程

10. **猫源Open格式兼容性**
   - 猫源Open格式编写的网页源存在兼容性问题，壳子适配暂未完全解决
   - 对写源感兴趣的用户请参考 `猫源open写网页源说明-适配壳子.md` 文档

## 📦 下载安装包

请从下方 Assets 中下载对应平台的安装包。

## ⚠️ 注意事项

- 请根据您的设备平台选择合适的安装包
- iOS用户如需切换nodejs接口，请务必按照上述步骤操作，避免应用闪退
- Windows 10用户如遇后台进程残留，请手动结束进程
